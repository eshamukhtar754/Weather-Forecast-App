# 🌤️ Weather Forecast App

A modern and responsive Weather Forecast application built with HTML, CSS, and JavaScript using the OpenWeatherMap API. Search any city in the world and get real-time weather information along with a 5-day forecast.

## ✨ Features

- 🔍 Search weather by city name
- 🌡️ Current temperature
- ☁️ Weather condition and icon
- 💧 Humidity
- 🌬️ Wind speed
- 👀 Visibility
- 🌍 Country and city information
- 📅 5-Day Weather Forecast
- ❌ Error handling for invalid city names
- 📱 Responsive design

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- OpenWeatherMap API

## 🚀 How to Run

1. Open the project folder.

3. Add your OpenWeatherMap API Key in `script.js`.

```javascript
const API_KEY = "YOUR_API_KEY";
```

4. Run `index.html` in your browser.
